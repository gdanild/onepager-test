# Personal One-Pager Website

## Overview
A professional one-page profile website for Danila Grischenko, showcasing experience, skills, and contact information with a "quiet luxury" minimal design aesthetic.

## Project Structure
- `server.js` - Node.js static file server (serves on port 5000)
- `public/` - Static website files
  - `index.html` - Main HTML page
  - `styles.css` - CSS styling (minimal, light theme)
  - `script.js` - Scroll animations

## Running the Project
The server runs with `node server.js` on port 5000.

## Key Features
- Single-page responsive design
- Smooth scroll reveal animations
- Mobile-friendly layout
- Sections: Hero, Value, Experience, Skills, Education, Contact
